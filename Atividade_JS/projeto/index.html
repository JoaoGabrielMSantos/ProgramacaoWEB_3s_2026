Instruções: Clique nos botões abaixo para executar cada programa individualmente, ou use o botão "Executar Todos em Sequência" para rodar todos de uma vez. Os resultados aparecerão no console abaixo.

1. Jogo de Adivinhação

Adivinhe o número secreto entre 1 e 20. O programa dará dicas se o número é maior ou menor.

2. Pedra, Papel ou Tesoura

Jogue contra o computador. Escolha entre pedra, papel ou tesoura.

3. Tabuada

Veja a tabuada de qualquer número de 1 a 10.

4. Triângulo de Asteriscos

Desenhe um triângulo com asteriscos. Escolha o número de linhas.

5. Soma da Série (1 + 11 + 111 + ...)

Calcule a soma da série com N termos (ex: 1 + 11 + 111 + 1111).
